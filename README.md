# 🔐 CIPHER DEFENSE

Cyberpunk hacker defense game with cipher training.

## How to Play

1. **Select ciphers** you want to train (Morse, Pigpen, Braille, Grid, Phone T9, Semaphore)
2. **Type the letter** that matches the cipher on each incoming threat to neutralize it
3. **Build combo** by chaining correct decodes
4. **Survive 5 stages** by not letting threats reach your firewall

## Features

- 6 cipher types (toggleable in selection)
- Mistake-weighting (weak letters repeat more often)
- Per-cipher accuracy stats at the end
- Cyberpunk visuals with matrix code rain, neon glow, particles
- Web Audio API sound effects
- Save best score to localStorage

## Tech

- Pure HTML + CSS + JavaScript (Canvas-based)
- Single-file deployment
- No external dependencies
